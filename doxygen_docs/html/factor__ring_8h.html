◆ is_irreducible()

template<typename T>
bool is_irreducible ( const Polynomial< T > & poly)

Checks if a polynomial is irreducible over a finite field Z_p.

Параметры шаблона
TThe coefficient type, expected to be a ModInt<P> for some prime P.
Аргументы
polyThe polynomial to check.
Возвращает
true if the polynomial is irreducible, false otherwise. The check involves testing divisibility by all possible polynomials of degree up to deg(poly)/2. Assumes the coefficient type T provides MOD_VALUE static member.

◆ run_factor_ring()

template<int P>
void run_factor_ring ( )

Runs an interactive session for performing operations in a factor ring Z_P[x]/(f(x)).

Параметры шаблона
PThe prime modulus for the finite field Z_P. Prompts the user to enter an irreducible polynomial f(x) and two elements (polynomials) a(x), b(x). Then performs a selected arithmetic operation (addition, subtraction, multiplication, division, inversion, exponentiation) on the elements a(x) and b(x) within the factor ring defined by f(x).
