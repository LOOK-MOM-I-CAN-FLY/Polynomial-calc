Подробное описание

template<typename T>
class FactorRingElement< T >

Template class representing an element in a factor ring F[x]/(f(x)).

Параметры шаблона
TThe type of the coefficients of the polynomials (e.g., ModInt

). An element is represented by a polynomial reduced modulo another polynomial (the ideal).
