Подробное описание

template<typename T>
class Polynomial< T >

Template class for representing a polynomial with coefficients of type T.

Параметры шаблона
TThe type of the coefficients (e.g., int, double, ModInt). The polynomial is stored as a vector of coefficients where coeffs[i] corresponds to x^i.
